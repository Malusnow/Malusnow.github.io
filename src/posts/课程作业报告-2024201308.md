## 二、作业要求理解

根据题目要求，本次作业需使用 AI Coding 工具/IDE、数据库（MySQL 或国产数据库/MongoDB）开发一个软件系统，至少实现 6-7 个功能模块，并在报告中给出功能说明、界面截图、核心代码片段。

本项目选型如下：

- IDE：Visual Studio Code
- 全栈语言：JavaScript
- 数据库：MySQL（使用 Docker 部署，不在本机直接安装）
- 后端：Node.js + Express
- 前端：React + Vite
- AI 功能：调用 OpenAI 兼容接口，生成学习计划

## 三、系统架构设计

### 3.1 架构图（文字说明）

1. 前端 React 页面发起 HTTP 请求。
2. 后端 Express 提供 RESTful API。
3. 后端通过 mysql2 连接 Docker 中的 MySQL。
4. 后端额外封装 AI 接口 `/api/ai/study-plan`。

### 3.2 数据库设计

数据库名：`course_tool_01308`

主要数据表：

1. `users_01308`：用户登录信息
2. `students_01308`：学生信息
3. `courses_01308`：课程信息
4. `enrollments_01308`：选课成绩信息
5. `tasks_01308`：任务看板
6. `announcements_01308`：公告信息

说明：表名统一带 `_01308`，用于满足“截图中体现学号后五位”要求。

## 六、项目运行步骤

1. 在项目根目录执行 `docker-compose up -d`（或 `docker compose up -d`）。
2. 进入 backend：`cp .env.example .env && npm install && npm run dev`。
3. 进入 frontend：`npm install && npm run dev`。
4. 浏览器访问 `http://localhost:5173`。
5. 使用 `admin_01308 / 123456` 登录。
